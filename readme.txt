== Description ==

IBN Shortcode adalah plugin untuk menyisipkan Iklan IDBlogNetwork pada Content Blog Wordpress anda.
Semoga dengan adanya Plugin ini, bisa memberikan Kemudahan bagi Blogger untuk memasang Iklan IDBlogNetwork pada Blog anda.

Untuk Penggunaanya Silahkan Kunjungi http://wan.web.id/plugins

== Installation ==

1. Upload Folder IBN-Shortcode ke folder wp-content/plugins/ 
2. Bisa juga upload IBN-Shortcode.zip lewat menu Plugins - Add New - Upload
3. Aktifkan Plugin

== Frequently Asked Questions ==

Untuk Penggunaan Plugin ini, Silahkan Sisipkan Shortcode dibawah ini pada Content Wordpress Anda..

Format Shortcode :

[ibn blogID="Blog ID IDBlogNetwork Anda" format="jenis iklan"]


Format :

PPA Static Banner
---------------------
- 120 x 600 = 1
- 160 x 600 = 2
- 300 x 250 = 3
- 468 x 60  = 4


PPC Flash Banner
---------------------
- 120 x 600 = 5
- 160 x 600 = 6
- 300 x 250 = 7
- 468 x 60  = 8
- 728 x 90  = 9


PPC Static Banner
---------------------
- 120 x 600 = 10
- 160 x 600 = 11
- 300 x 250 = 12
- 468 x 60  = 13
- 728 x 90  = 14
